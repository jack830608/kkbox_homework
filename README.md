# node.js + express + next.js + react + redux + styled-components

Requires [Node.js](https://nodejs.org/) v10+ to run. 

### For local development

Run with [nodemon](https://nodemon.io/)
```sh
$ cd project_folder
$ npm install
$ npm run dev
```

### For production

```sh
$ cd project_folder
$ npm install
$ npm run build
$ npm start
```
